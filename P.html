📡 Apa itu IP Address?

IP Address (Internet Protocol Address) adalah alamat unik yang diberikan ke setiap perangkat yang terhubung ke jaringan komputer (internet atau lokal).

🔹 Contoh: 192.168.1.1
🔹 Fungsi: Identifikasi perangkat & lokasi di jaringan
🔹 Format: 4 angka desimal (0–255) dipisah titik
🖥️ ➡️ 🌍

📊 Mengapa Ada Kelas IP?

Agar alamat IP tidak boros dan mudah dikelola. Dibagi menjadi 5 kelas: A, B, C, D, E berdasarkan bit awal.

🧩 Penjelasan Tiap Kelas + Contoh

🇦 Kelas A
Network ID panjang, Host ID besar.
📌 Contoh: 8.8.8.8 (Google DNS)
🇧 Kelas B
Seimbang untuk menengah.
📌 Contoh: 172.16.0.1 (jaringan kampus)
🇨 Kelas C
Banyak network, sedikit host.
📌 Contoh: 192.168.1.1 (router rumahan)
🇩 Kelas D
Multicast (satu ke banyak).
📌 Contoh: 224.0.0.1 (semua host subnet)
🇪 Kelas E
Untuk riset, tidak dipakai publik.
📌 Contoh: 240.0.0.0 eksperimental
🏢🏥🏠 🎥 🔬

✅ Kesimpulan

  • IP Address adalah identitas unik perangkat di jaringan.
  • IP Class A, B, C dipakai sehari-hari (komersial).
  • Kelas D untuk multicast, Kelas E untuk eksperimen.
  • Saat ini sudah berkembang Classless (CIDR) → lebih fleksibel tanpa terikat kelas A/B/C.
  • Namun memahami IP Class penting untuk memahami sejarah & dasar jaringan.
